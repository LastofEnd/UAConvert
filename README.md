# UA Tools

**UA Tools** — невелика утиліта для Android, написана мовою Kotlin.

Застосунок додає команди до стандартного меню Android **«Обробити текст»**. Вони дозволяють перетворювати виділений текст між українською та англійською розкладками клавіатури, замінювати дефіс на довге тире, а також додавати або прибирати наголос.

## Можливості

- Перетворення виділеного тексту між розкладками UA ↔ EN
- Автоматичне визначення напрямку перетворення
- Збереження регістру літер
- Заміна `-` на `—`
- Додавання та видалення наголосу
- Робота через системне меню виділеного тексту
- Відсутність доступу до інтернету
- Відсутність дозволів на файли, контакти, повідомлення або місцезнаходження

## Як користуватися

1. Виділіть текст у застосунку, який підтримує стандартну функцію Android **«Обробити текст»**.
2. Відкрийте додаткове меню для виділеного тексту.
3. Оберіть одну з команд:
   - **UA↔EN** — перетворити текст між українською та англійською розкладками;
   - **Наголос ´** — додати або прибрати наголос.
4. Виділений текст буде замінено результатом.

### Приклади

```text
ghbdtn → привіт
руддщ → hello
- → —
а → а́
а́ → а
```

> Доступність пункту «Обробити текст» залежить від конкретного застосунку та версії Android.

## Вимоги

- Android 12 або новіша версія
- Мінімальний рівень API: 31

## Встановлення

Завантажте APK із розділу **Releases** на GitHub:

> GitHub → Releases → Latest release → завантажити `.apk`

Після завантаження відкрийте APK та дозвольте встановлення застосунків із цього джерела, якщо Android попросить це зробити.

## Збірка з вихідного коду

Для збірки потрібні:

- Android Studio
- Android SDK 36
- JDK 11 або новіша сумісна версія

### Через Android Studio

1. Відкрийте папку проєкту в Android Studio.
2. Дочекайтеся завершення синхронізації Gradle.
3. Оберіть:

```text
Build → Build App Bundles or APKs → Build APKs
```

Debug APK буде створено приблизно за шляхом:

```text
app/build/outputs/apk/debug/app-debug.apk
```

### Через командний рядок

Windows:

```powershell
.\gradlew.bat assembleDebug
```

Linux або macOS:

```bash
./gradlew assembleDebug
```

Для release-збірки:

```powershell
.\gradlew.bat assembleRelease
```

Release APK потрібно підписати власним ключем перед розповсюдженням.

## Структура роботи

Застосунок містить дві дії для системного `PROCESS_TEXT`:

- `ProcessConvertActivity` — перетворення розкладки та заміна дефіса;
- `ProcessAcuteActivity` — додавання або видалення комбінованого знака наголосу.

Головна активність не має окремого інтерфейсу, оскільки основна взаємодія відбувається через меню виділеного тексту Android.

## Конфіденційність

Застосунок обробляє лише текст, який користувач сам виділив і передав через системну команду Android. У маніфесті немає дозволу на доступ до інтернету.
